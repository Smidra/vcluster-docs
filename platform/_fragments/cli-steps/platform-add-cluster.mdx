    **Add Host Cluster to the Platform (Optional)**:Add the cluster to the platform. If this cluster is already to the platform, then you can skip to the next step.

    ```
    vcluster platform add cluster <CLUSTER_ID> --display-name <UI_CLUSTER_NAME>
    ```

    Replace the following:

    * `<CLUSTER_ID>`: ID of the host cluster.  
    * `<UI_CLUSTER_NAME>`: Name of your host cluster.
