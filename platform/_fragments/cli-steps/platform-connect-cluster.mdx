    **Connect to the host cluster**: You can only connect to host clusters that have already been added to the platform.

      ```
      vcluster platform connect cluster <UI_CLUSTER_NAME>
    ```

    Replace the following:

    *  `<UI_CLUSTER_NAME>`: Name of your host cluster. 
<p></p>

:::info
When connecting to a host cluster, the kube-context automatically gets updated to the host cluster.
:::
